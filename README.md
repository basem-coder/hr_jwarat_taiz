# 💎 نظام إدارة جوهرة تعز مول (الخزينة و HR) 
### Johara Taiz Mall Management System (Treasury & HR)

نظام متكامل ومؤتمت مصمم خصيصاً لإدارة العمليات المالية والموارد البشرية في **جوهرة تعز مول**. يهدف النظام إلى تعزيز الرقابة المالية، وتنظيم صرف الرواتب والسلف، وتوفير تقارير دقيقة للإدارة العليا.

---

## 🚀 المميزات الرئيسية (Core Features)

### 💰 وحدة إدارة الخزينة (Treasury Module)
* **تسجيل الحركات:** رصد الإيرادات والمصاريف اليومية وتوريدات الإدارة.
* **الرقابة الفورية:** تحديث تلقائي لصافي العهدة النقدية في الصندوق.
* **التوثيق:** ربط العمليات المالية ببيانات الموظفين (السلف).

### 👥 وحدة الموارد البشرية (HR & Payroll)
* **إدارة الموظفين:** تصنيف الموظفين حسب الأقسام (الإدارة، صالة الألعاب، الأمن، النظافة).
* **نظام السلف الذكي:** خصم تلقائي للسلف من الراتب الأساسي عند توليد الكشوفات.
* **تقييم الأداء:** نظام تقييم شهري للموظفين لضمان جودة العمل.

### 📊 التقارير والطباعة (Reporting)
* **تصدير Excel:** كشوفات رواتب تفصيلية لكل قسم على حدة.
* **طباعة PDF:** تقارير رسمية جاهزة للطباعة تدعم اللغة العربية.
* **إشعارات WhatsApp:** إرسال تفاصيل الراتب والصافي للموظفين آلياً.

### 🔐 الأمان والنسخ الاحتياطي (Security)
* **نظام دخول:** حماية النظام بكلمة مرور للمسؤول فقط.
* **Auto-Backup:** نسخة احتياطية تلقائية على **Google Drive** لضمان عدم ضياع البيانات.
* **قاعدة بيانات محلية:** استخدام SQLite لضمان سرعة واستقرار البيانات.
