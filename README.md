## 📝 نبذة عن المشروع
هذا المشروع عبارة عن نظام متكامل لإدارة العمليات التشغيلية والمالية في **مول جوهرة تعز**. تم تطويره لرقمنة العمليات اليدوية وتحويلها إلى نظام آلي دقيق يسهل عملية الرقابة واتخاذ القرار.

## 🚀 الميزات الرئيسية
* **إدارة المستأجرين:** تتبع بيانات 56 مستأجر مع سجلات دقيقة لكل وحدة تجارية.
* **نظام التحصيل المالي:** أتمتة تحصيل (الإيجارات، رسوم الكهرباء، رسوم المياه).
* **تقارير أوتوماتيكية:** توليد تقارير PDF دورية وإرسال إشعارات عبر WhatsApp.
* **إدارة الموارد البشرية:** مراقبة أداء الموظفين وتنظيم الكادر الإداري.
* **واجهة مستخدم (GUI):** تم بناؤها باستخدام مكتبة `Tkinter` لتسهيل التعامل مع البيانات.

## 🛠 الأدوات والتقنيات المستخدمة
* **Python:** اللغة الأساسية لبناء النظام الإداري.
* **Excel VBA:** لأتمتة العمليات الحسابية المتقدمة وتوليد الكشوفات.
* **Tkinter:** لتصميم واجهات البرنامج المكتبية.
* **SQLite/Excel:** لقواعد البيانات وتخزين السجلات.
